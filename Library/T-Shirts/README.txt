# Bonus Mockup Usage Guide

## Congratulations on choosing Bulk Mockup Master! 

To get you started immediately, we have included these 3 Premium, Royalty-Free Base Mockups.

### How to use:
1. Open the App.
2. Click **"Upload Base Image"**.
3. Navigate to this folder (`Bonus Mockups`) and select one of the images (e.g., `Flat Lay.png`).
4. Select your **Design Folder** containing your artworks.
5. Click **Generate All**!

These images are provided Royalty Free for use with this software. You can use them for commercial projects, listings, and marketing materials.
